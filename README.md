# Expense_Tracker_-CSV-JSON-
A menu-driven Expense Tracker built using Python and CSV file handling. Users can add, view, calculate, and delete expenses while maintaining a neatly organized expense record.
Stores expense records in a CSV file that can be opened and edited in Microsoft Excel.

#Features
• Add new expenses
• Categorize expenses (Food, Travel, Study, Other)
• View saved expenses
• Calculate total spending
• Find highest and lowest expenses
• Delete unwanted expense records
• Store all data in CSV format

#Outcomes
• Working with Python dictionaries and lists
• Reading and writing CSV files
• Building menu-driven console applications
• Performing data analysis using loops
• File handling in Python
